## ğŸŒ About This Project

The Wildlife Economy Investment Index (WEII) Platform is a comprehensive web application that presents a framework for integrating biodiversity metrics into sovereign debt instruments. This capstone project demonstrates how WEII can evolve from a diagnostic tool into an investable mechanism that aligns fiscal incentives with conservation outcomes.

### Key Features

- **Interactive Bond Calculator** - Model fiscal impacts of WEII-linked sovereign bonds
- **Country Analysis** - Deep comparison of Tanzania & South Africa readiness
- **Benchmark Deals** - Analysis of Rhino Bond, Gabon, Ecuador, Uruguay transactions
- **Five-Pillar Framework** - Legal, Institutional, Market, MRV, and Roadmap integration
- **Risk Matrix** - Comprehensive risk register with mitigation strategies
- **Market Data** - Latest 2025 biodiversity finance statistics
